# 基于FPGA的超宽带脉冲星信号信道化算法研究
摘要

随着射电天文观测技术的飞速发展，超宽带（Ultra-Wideband, UWB）信号处理已成为下一代大口径射电望远镜数字终端系统的核心挑战。针对脉冲星观测中海量数据流对高实时性、高处理精度及有限存储资源的严苛需求，研究高效、灵活的信号信道化（分通道）算法及其硬件实现技术具有极其重要的学术意义与应用价值。

本文紧密结合未来大口径射电望远镜超宽带接收系统的发展方向，系统性地研究了基于现场可编程门阵列（FPGA）的超宽带信号信道化理论与实现方法。针对硬件开发过程中传统逻辑描述语言（HDL）编程效率低、开发周期长等痛点，提出并实现了基于高层次综合（Vitis HLS）的高性能有限冲激响应（FIR）滤波器硬件加速方案。为解决超宽带信号在传统信道化过程中存在的混叠失真与数据冗余矛盾，创新性地提出了一种有理数倍过采样（Rationally Oversampled）信道化算法，在保障信号重构精度的同时，有效平衡了数据传输带宽与存储压力。以异构集成平台射频系统级芯片（RFSoC）为基础，设计并构建了一套完整的超宽带脉冲星信号分通道处理与实时传输流水线，实现了从射频直接采样、子带划分到万兆以太网数据封装的一体化处理。通过仿真实验及南山26米射电望远镜的实测数据验证，本文所设计的算法及系统在时序性能、资源利用率及天文科学产出方面均表现优异。

本文的主要研究内容与成果如下：

（1）针对超宽带信号处理中高性能滤波器的硬件实现问题，研究了基于 Vitis HLS 的高层次综合优化机制。设计了 127 阶 Hamming 窗直接型 FIR 滤波器 IP 核，通过流水线打拍与循环回绕等技术策略，显著提升了硬件开发效率。对比实验表明，与传统 Verilog HDL 实现方式相比，HLS 方案在保持相近资源利用率与时序性能的前提下，极大缩减了代码规模与工程开发周期，为复杂天文信号处理算法的快速硬件部署提供了技术支撑。

（2）针对传统信道化算法在精度与带宽利用率之间的失衡问题，提出了一种有理数倍过采样信道化算法。通过引入分数倍过采样因子，结合多相滤波器组、循环移位器及裁剪器结构，实现了对子带带宽与采样率的灵活配置。该算法有效解决了临界采样引发的子带边沿失真以及整数倍过采样导致的数据量激增问题。利用 Python 对脉冲星 J0437−4715 实测数据进行处理，验证了算法在保持信号完整性与提升计算灵活性方面的有效性。

（3）设计并实现了基于 RFSoC 的超宽带脉冲星信号处理流水线。针对 RFSoC 平台的异构特性，利用其片上高速 ADC 实现了超宽带信号的直接采样；在逻辑端通过引入 Winograd 快速傅里叶变换（WFFT）与质因数映射算法优化了 20 点并行 DFT 模块，显著降低了 FPGA 资源消耗。同时，设计了基于 VDIF 协议的以太网传输模块，实现了海量天文数据的高效实时流转。

（4）完成了超宽带信道化系统的科学验证与性能评估。基于南山 26 米射电望远镜 L 波段接收机开展了实测实验，成功记录并处理了 J0332+5434、J1935+1616 及 J2022+2854 等多颗脉冲星的观测数据。实验结果显示，处理得到的动态谱与脉冲轮廓清晰对齐，子带合成结果符合科学预期，充分证明了本文设计的超宽带信道化算法及其硬件流水线在实际天文观测任务中的可靠性与实用性。

论文的主要创新点如下：

（1）创新性提出了有理数倍过采样信道化算法。突破了传统倍数限制，实现了采样率与子带带宽的精细化匹配，在保障射电天文科学精度的前提下，有效缓解了超宽带数据流的存储与传输瓶颈。

（2）优化并实现了基于 WFFT 的高通量 DFT 硬件架构。针对 20 点等特殊长度的变换需求，通过算法映射与逻辑复用，在 FPGA 上实现了低复杂度、高并行度的实时频谱转换，提升了超宽带数字终端的实时处理能力。

（3）构建了基于 RFSoC 的全链路超宽带天文信号处理流水线。 利用 SoC 异构集成优势，实现了包含直接射频采样、有理倍数信道化以及标准 VDIF 数据封装的单片化处理架构。该方案在保障高处理带宽的同时，极大优化了系统复杂度与功耗，为 QTT 等未来大口径望远镜超宽带数字终端的研制提供了高效的参考模型。
